
## Instalasi

- download menggunakan terminal di "https://github.com/irulhuda983/siakadTI.git"
- jalankan web server dan dbms
- siapkan data base
- masuk ke project siakadTI
- copy file env.example ubah menjadi .env
- ubah user, password, port, dan database
- jalankan perintah "composer install"
- jalankan perintah "php artisan key:generate"
- jalankan perintah "php artisan migrate"
- jalankan peiinah 'php artisan serve"
- buka chrome dan jalankan "http://127.0.0.1:8000"
